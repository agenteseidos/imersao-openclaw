# Imersão OpenClaw: Agentes em Ação

Página de vendas da imersão — `imersao.agenciaeidos.com.br`

## Deploy na Vercel

1. Autenticar: `vercel login`
2. Deploy: `cd projects/imersao-openclaw && vercel --yes --prod`
3. Adicionar domínio: `vercel domains add imersao.agenciaeidos.com.br`

O DNS CNAME já está configurado apontando para `cname.vercel-dns.com`.
